Color

32-bit Color with alpha channel.

Module require("gui")
Type Integer

Color is represented by a 32-bit ARGB integer.

For APIs that accept Color, you can pass a hex string representing the color, like "#A123", "#123", "#AA112233", "#112233".

label.setColor('#F000')
